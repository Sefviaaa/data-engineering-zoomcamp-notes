# 03_data_warehouse - Data Warehouse with BigQuery

This module covers data warehousing concepts and hands-on implementation using Google BigQuery, as part of the Data Engineering Zoomcamp.

## Contents

| Folder | Description |
|--------|-------------|
| `Homework/` | Homework solutions with Airflow DAGs for data ingestion |

## Topics Covered

- **BigQuery Fundamentals**: Tables, partitioning, clustering
- **Data Ingestion**: Loading data from GCS to BigQuery
- **SQL in BigQuery**: Querying large datasets efficiently
- **Cost Optimization**: Partitioning and clustering strategies
